# vue-manager
基于vue的管理系统，包含大数据看板，流程管理，工单管理，权限，主题切换等等功能。项目持续更新中


## Project setup 依赖包安装
```
npm install
```

### Compiles and hot-reloads for development 开发环境编译运行
```
npm run serve
```

### Compiles and minifies for production 编译打包上线
```
npm run build
```

### Lints and fixes files 格式化，修复
```
npm run lint
```
